# Advent of Code 2025

My fancy struggling and solutions for [Advent of Code 2025](https://adventofcode.com/2025) in Go.

## About

Advent of Code is an annual programming challenge created by [Eric Wastl](https://twitter.com/ericwastl). Please support his work!

## Usage
```bash
go run main.go
```

## Note

Puzzle inputs are not included per AoC guidelines. Get yours at [adventofcode.com](https://adventofcode.com/2025).
